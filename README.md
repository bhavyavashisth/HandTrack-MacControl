# Gesture-control
Cyberpunk Gesture Control v4

A macOS-optimized gesture interface with a sleek HUD.
🚀 Setup Instructions

    Python Version: Ensure you use Python 3.11 (MediaPipe does not support 3.14 yet).

    Environment:
    Bash

    python3.11 -m venv gesture_env
    source gesture_env/bin/activate

    Install Dependencies:
    Bash

    pip install opencv-python mediapipe pyautogui numpy

    Permissions: You must grant Camera and Accessibility access to your Terminal in System Settings > Privacy & Security for the cursor to move.

🎮 Gestures

    Index Finger: Move Cursor

    Pinch: Left Click

    Fist to Palm: Maximize Window

    Palm to Fist: Quit App

    3 Fingers: Right Click
